# EXT2-Linux-File-System
EXT2 file system emulator. It works like Linux does.<br>

# Action
<img src="image/run.png" width="60%" height="60%">

## Run

```sh
$ make
```

```sh
$ ./shell
```

```sh
$ [/]# format
```

```sh
$ [/]# mount
```

After this commands you can use several Linux command like touch, mkdir, mv, ls, cd...

## Comment
